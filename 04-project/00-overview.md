# Project Configuration

Every skillbook is a standard npm project. This section covers the package manifest,
versioning policy, and how metered pricing works.

## When to Read This Section

- You're setting up a new skillbook and need to configure `package.json`
- You want to understand how versioning affects billing
- You need to set the right price for your content
- You want to understand the sync rules between `package.json` and `SKILL.md`

## Pages in This Section

- `01-package-json.md` — package.json structure, the `skillbook` key, and sync rules
- `02-versioning.md` — Semver policy and how version changes affect metering
- `03-pricing.md` — Metered per-page billing and pricing tiers

Read `01-package-json.md` first — it's the foundation. Versioning and pricing build on it.

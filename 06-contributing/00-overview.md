# Contributing

The Skillbook Format is an open specification. This section covers how to suggest changes,
report issues, and contribute improvements.

## When to Read This Section

- You have an idea for improving the format
- You found an ambiguity, contradiction, or error in the spec
- You want to submit a change via pull request

# Contributing

See [07-contributing/01-how-to-contribute.md](07-contributing/01-how-to-contribute.md) for the full guide.

**Quick version:**

1. **Idea?** Start a [Discussion](https://github.com/skillbooks-ai/skillbook/discussions)
2. **Bug?** Open an [Issue](https://github.com/skillbooks-ai/skillbook/issues)
3. **Fix?** Submit a PR referencing the Issue or Discussion
